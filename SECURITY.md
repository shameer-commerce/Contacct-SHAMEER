# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported          |
| -------- | ------------------ |
| 5.1.x    | ✅                 |
| 5.0.x    | ❌                 |
| 4.0.x    | ✅                 |
| < 4.0    | ❌                 |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

- **Email:** security@yourdomain.com  
- **Response time:** We aim to respond within 3 business days.  
- **Process:**  
  1. We will confirm receipt of your report.  
  2. We will investigate and provide status updates.  
  3. Once resolved, we’ll credit you (if desired) and release a fix.

Please **do not publicly disclose** the issue until we’ve addressed it.
